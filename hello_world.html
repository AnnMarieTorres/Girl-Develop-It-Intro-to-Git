Version control allows you and your team to do several powerful things: collaborate, track, and revert changes.

Git was created in 2005. It is fast, distributed, each commit has a corresponding hash, and everyone has a local copy of the history.

  • Installation and Setup:
      Make sure you have iterm or a similar commmand line tool.
      Install git here: http://git-scm.com/
      Configure your username and email, create your public/private keys.

    You are now ready to create your first repo!

  • To creat a repo:
      • mkdir Name_of_folder
      cd Name_of_folder
      git init

    That's it, that's all there is to "git-ing" set up. Hehehe.
